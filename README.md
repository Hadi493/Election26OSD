# Polling Centers

Opensource Data for Polling Centers in Bangladesh.

Data scrapeed from [Bangladesh Election Commission](https://ecs.gov.bd/polling-station).

![D1 Preview](screenshot.png)

## Quick start:

```bash
git clone https://github.com/Hadi493/Election26OSD.git
cd Election26OSD
```

Open two terminal and run frontend and backend.

```bash
cd Election26OSD/frontend/
npm install
npm run start
```

```bash
cd Election26OSD/backend/
npm install
npm run start
```
